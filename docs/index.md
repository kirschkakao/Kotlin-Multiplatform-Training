# Welcome to the Kotlin Multiplatform Training

This training introduces you to the development of Kotlin Multiplatform (KMP) applications, with a practical focus on building a Carcassonne game. Over 5 days, you will learn fundamental concepts of Kotlin, Jetpack Compose, Material Design 3, navigation graphs, and manual dependency injection while gradually developing a complete game.

!!! note
    Make sure you have installed the required tools (see Day 1). The code will be provided in a separate repository, and you will check in your progress there.
